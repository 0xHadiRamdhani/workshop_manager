# Screenshots

Tempat untuk menyimpan screenshot aplikasi Workshop Manager.

## Cara Mengambil Screenshot

### iOS Simulator
1. Jalankan aplikasi di iOS Simulator
2. Tekan `Cmd + S` untuk mengambil screenshot
3. Screenshot akan tersimpan di desktop

### Android Emulator
1. Jalankan aplikasi di Android Emulator
2. Klik tombol camera di sidebar emulator
3. Screenshot akan tersimpan di desktop

### Desktop (Windows/macOS/Linux)
1. Jalankan aplikasi di desktop
2. Gunakan tombol screenshot bawaan sistem operasi
3. Simpan screenshot di folder ini

## Nama File yang Direkomendasikan

- `dashboard.png` - Screenshot halaman dashboard
- `workshop.png` - Screenshot halaman workshop
- `cashier.png` - Screenshot halaman kasir
- `about.png` - Screenshot halaman about
- `add_vehicle.png` - Screenshot form tambah kendaraan
- `transaction_history.png` - Screenshot riwayat transaksi

## Ukuran yang Direkomendasikan

- iPhone: 390x844 (iPhone 14 Pro)
- iPad: 768x1024
- Android: 360x640
- Desktop: 1280x720

Catatan: Ganti file ini dengan screenshot aktual aplikasi.
